Landing page pessoal desenvolvida com HTML5, CSS3 e JavaScript puro — sem frameworks. Dark mode com estética tech, animações suaves e design responsivo para mobile e desktop.

## ✨ Funcionalidades

- **Efeito de digitação** no Hero com frases rotativas
- **Partículas animadas** no fundo com Canvas API
- **Scroll-spy** — o menu destaca a seção ativa automaticamente
- **Menu hamburger** para mobile
- **Animações de entrada** com Intersection Observer
- **Barras de habilidade** animadas
- **Formulário de contato** com feedback visual
- **Dark theme** com variáveis CSS

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 semântico | Estrutura da página |
| CSS3 + Variáveis | Estilo, Grid, Flexbox e responsividade |
| JavaScript Vanilla | Lógica e animações |
| Canvas API | Partículas animadas |
| Intersection Observer | Animações de scroll |
